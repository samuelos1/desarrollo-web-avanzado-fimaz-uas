# Proyecto Final – Desarrollo Web Avanzado

## Objetivo
Desarrollo de aplicación web completa utilizando MVC y API REST.

## Tecnologías utilizadas
- PHP 8+
- MySQL
- MVC
- REST
- JSON

## Instrucciones de ejecución
1. Importar base de datos (.sql).
2. Configurar conexión.
3. Ejecutar en servidor local.
4. Probar API con Postman.
